# Formatação dos Dados

_Origem: `Formatação dos Dados` na collection Postman._

Em algumas partes da documentação (principalmente em Objetos) apontamos o tipo dos dados para determinada informação. A seguir estão os padrões para tais tipos

| Tipo de dado | Exemplo | Descrição |
| --- | --- | --- |
| Inteiro | 99 | Números inteiros |
| String | "Este é um texto" | Qualquer tipo de texto sem formato |
| Decimal | "99.90" | Número decimal com até duas casas decimais |
| Date | 2022-07-13 | Data no formato ISO |
| Datetime | 2022-07-13T12:59:36.759399+00:00 | Data e hora no formato ISO, sempre no fuso horário UTC (+00) |
